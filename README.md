## Dependencies
1. PC/Laptop with NodeJS installed.
2. Arduino Uno/Mega.
3. DS18B20 Waterproof temperature sensor.
4. 5v Relay.

## Installation
1. Clone repository
2. Run: ```npm install```
3. Run ```node index.js```
4. Open http://127.0.0.1:4200

## Configuration
Look at "./conf.json".
